## Using the Notebooks

1. Clone the repository:
   ```bash
   git clone https://github.com/JarvisSan22/Python_101.git
   ```

2. Install Jupyter:
   ```bash
   pip install jupyter
   ```

3. Launch Jupyter:
   ```bash
   jupyter notebook
   ```

4. Navigate to the notebooks directory and open the desired notebook
